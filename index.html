First you need:

  • define a grid container by adding a display property to it with a grid value;
  • set the sizes of rows and columns using the grid-template-rows and grid-template-columns properties;
  • put children in the grid using the grid-row and grid-column properties.

With repeat(): grid-template-columns: repeat(3, 8em);
